Product Description

SecureBridge Components library offers set of nonvisual components for Delphi and C++Builder. They provide support for secure connection with remote services using SSH or SSL protocol. These protocols serve for establishing secure connections between computers through public networks.

SSH and SSL are secure low-level transport protocols, that provides stable encryption, different authenticaton methods for both client and server and built-in protection. These protocols can replace existing TCP/IP protocols such as Telnet. Both client and server must support SSH/SSL for establishing secure connection.

To establish secure connection using SecureBridge components it's enough to select required authentication methods and encrypting methods, client or server address. After that connection is established.

Installation

Compiled versions of SecureBridge are installed automatically by the SecureBridge Installer for all supported IDEs. Versions of SecureBridge with Source Code must be installed manually. For more information, see the Installation topic of the SecureBridge Help documentation.

Documentation and informational resources

SecureBridge comes with full documentation, which includes a detailed reference for all the SecureBridge components and classes and set of articles that highlight SecureBridge advanced functionality. The SecureBridge documentation is integrated into your IDE at install time. It can be invoked by pressing F1 in an object inspector or on a selected code segment.

If you have a question regarding SecureBridge, visit the SecureBridge forum online at http://forums.devart.com/.

If your problem is not covered in the documentation or forum, see the support page for information on how to contact the SecureBridge support team.
