1. Because FlexCompress includes strong encryption, there are US export
restrictions on all products of US companies that use it. Does FlexCompress include compiler
directives that can exclude the encryption code so that compiled software can
be distributed without any restrictions?

Yes, FlexCompress has {$DEFINE ENCRYPTION} compiler directive which could be switched off.


2. How can I prevent other FlexCompress users to see the list of files inside of my archive?

With the source code version of FlexCompress, you can easily change FlexCompress file
signature constant, so other FlexCompress users will not be able to open your archive and to see the list of the compressed files.
