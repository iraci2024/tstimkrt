Describes a phase of current archive operation.

Unit
FlexCompress

type TFXCProgressPhase = (ppStart, ppProcess, ppEnd);

Description

TFXCProgressPhase value specifies phase of current archive operation.

TFXCProgressPhase value can be set to any of the following values:
Value
Meaning
ppStart
This value corresponds to the start phase of the operation
ppProcess
This value corresponds to the middle phase of the operation
ppEnd
This value corresponds to the end phase of the operation
