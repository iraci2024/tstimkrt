Specifies the size of the multi-spanning archive volumes.

Unit
FlexCompress

type TFXCVolumeSize = (vsAutoDetect, vsCustom, vs1_44MB, vs100MB, vs200MB, vs600MB, vs650MB, vs700MB, vs4700MB);

Description

TFXCVolumeSize value specifies the size of each multi-disk archive volume. Set SpanningOptions.VolumeSize to vsAutoDetect for automatic detection of the volume size (default value). You can also use vsCustom value and specify the volume size by setting SpanningOptions.CustomVolumeSize.


SpanningOptions.VolumeSize can be set to any of the following values:
Value
Meaning
vsAutoDetect
Autodetect of the volume size
vsCustom
Custom volume size
vs1_44MB
Volume size is 1.44 Mb (3.5" floppy disk)
vs100MB
Volume size is 100 Mb (IOMEGA disk)
vs200MB
Volume size is 200 Mb (MO disk)
vs250MB
Volume size is 250 Mb (MO disk)
vs600MB
Volume size is 600 Mb (MO disk)
vs650MB
Volume size is 650 Mb (74 Min. compact disk)
vs700MB
Volume size is 700 Mb (80 Min. compact disk)
vs4700MB
Volume size is 4700 Mb (DVD disk)
