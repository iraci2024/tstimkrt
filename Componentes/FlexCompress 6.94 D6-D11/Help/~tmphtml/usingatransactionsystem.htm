Using a transaction system

Previous  Top  Next


Use BeginUpdate, CancelUpdate, EndUpdate methods and InUpdate property of TFlexCompress for optimizing speed of executing the archive updating operations. Another great advantage of FlexCompress transaction system is possibility of cancelling all archive updating operations executed after calling BeginUpdate. This approach allows to update archive providing data integrity like reliable database systems.
