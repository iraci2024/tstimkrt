Occurs when disk when archive is created is full.

type TFXCOnDiskFullEvent = procedure(
Sender:     TObject;
VolumeNumber: Integer;
VolumeFileName: WideString;
var Cancel:     Boolean
) of object;
property OnDiskFull: TFXCOnDiskFullEvent;

Description

Write an OnDiskFull event handler to ask user to insert disk with sufficient free space for writing a multi-disk spanned archive, or to delete some files on the disk where archive is created.
VolumeNumber specifies a number of the volume that should be inserted. For single-volume archives VolumeNumber is 0.
Also you can override default file name for this volume using VolumeFileName. Use Cancel to abort current operation.
