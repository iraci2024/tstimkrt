Creating, opening and closing archive

Previous  Top  Next


Use OpenArchive, CloseArchive methods and FileName, Active properties of TFlexCompress to create, open or close an archive file.

Open archive file before using it. Do not forget to close archive when your application does not needs an access to the archive file. This allows other applications to use the archive file.

You should not update single archive file from multiple applications simultaneously. Use fmOpenReadWrite or fmShareExclusive mode to open archive file for updating.
