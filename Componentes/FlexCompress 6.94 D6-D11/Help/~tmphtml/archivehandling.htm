Archive handling

Previous  Top  Next


Use AddFiles, ExtractFiles, DeleteFiles, MoveFiles, TestFiles, UpdateFiles methods property of TFlexCompress for handling files stored within the archive.
These methods supports wildcards in file names that allows to make a group operations on many files by single call.

FlexCompress provides also some additional methods for handling single file stored inside the archive: AddFromBuffer, AddFromStream, AddFromString, ExtractToBuffer, ExtractToStream, ExtractToString, RenameFile.

Open archive file before using these methods.
