Occurs when last volume should be inserted while extracting or testing a multi-volume archive.

type TFXCOnRequestLastVolumeEvent = procedure(
Sender:       TObject;
var VolumeFileName:    WideString;
var Cancel:       Boolean
) of object;
property OnRequestLastVolume: TFXCOnRequestLastVolumeEvent;

Description

Write an OnRequestLastVolume event handler to perform asking the user for inserting a last volume of the multi-volume archive.
See also OnRequestFirstVolume and OnRequestMiddleVolume events.
Also you can override default file name for this volume using VolumeFileName. Use Cancel to abort current operation.
