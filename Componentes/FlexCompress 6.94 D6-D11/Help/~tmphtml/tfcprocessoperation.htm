TFXCProcessOperation type
TFlexCompress
Top 


Describes an archive updating operation.

Unit
FlexCompress

type TFXCProcessOperation = (poAdd, poMove, poDelete, poUpdate, poExtract, poTest);

Description

TFXCProcessOperation value specifies current archive updating operation.

TFXCProcessOperation value can be set to any of the following values:



Value
Meaning
poAdd
This value corresponds AddFiles method
poMove
This value corresponds MoveFiles method
poDelete
This value corresponds DeleteFiles method
poUpdate
This value corresponds UpdateFiles method
poExtract
This value corresponds ExtractFiles method
poTest
This value corresponds TestFiles method
poRename
This value corresponds RenameFile method
poChangeAttr
This value corresponds ChangeFilesAttr method
poChangeComment
This value corresponds ChangeFilesComment method
poMakeSFX
This value corresponds MakeSFX method
